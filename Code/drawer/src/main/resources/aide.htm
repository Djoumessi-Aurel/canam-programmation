SHAPE   DRAWER

Aide

Pour zoomer/dézoomer

-  Maintenez la touche Ctrl enfoncée et scrollez avec la souris : vers le haut pour zoomer, vers le bas pour dézoomer.

-  Vous pouvez aussi vous servir de la barre d’outils ou de menus.

Pour dessiner une forme

-  Choisissez une forme (via le menu ou la barre d’outils) puis cliquez dans la zone de dessin pour la positionner. Vous pourrez ensuite entrer ses paramètres (couleur, etc.)

Pour sélectionner une forme :

-  Cliquez dessus. Une fois qu’une forme est sélectionnée, vous pouvez la déplacer, la modifier ou la supprimer.

-  Pour désélectionner, cliquez juste ailleurs ou appuyez sur Echap.

NB : Utilisez la touche Echap pour annuler une action que vous avez commencé.

 

Parcourez la barre de menus pour voir toutes les possibilités ! 🙂
